## Cara Pakai

1. Buka website di `https://blur.codeboost.my.id`
2. Tekan **Aktifkan Kamera**
3. Izinkan akses kamera di browser
4. Tunjukkan isyarat damai dengan dua jari
5. Layar akan diburamkan otomatis
